# 🚀 Smart Task Manager

A modern task management web application built with React.

## 🔗 Live Demo

[https://your-project-name.vercel.app](https://smart-task-manager-amber.vercel.app/)

## 📌 Features

* Add, edit, and delete tasks
* Mark tasks as completed or pending
* Filter tasks (All / Completed / Pending)
* Persistent storage using localStorage
* Clean and responsive UI

## 🛠️ Tech Stack

* React (useState, useEffect)
* JavaScript (ES6)
* CSS (custom styling)

## 🧠 Key Concepts Used

* Component-based architecture
* State management with hooks
* Props and unidirectional data flow
* Conditional rendering
* Event handling

## 📂 Project Structure

src/
├── components/
├── App.js
├── index.js

## ⚙️ Installation

```bash
git clone https://github.com/arnob56/smart-task-manager.git
cd smart-task-manager
npm install
npm start
```
